# Customer Behavior
This project is to answer the business question: 
**“How can the company leverage consumer shopping data to identify trends, improve
customer engagement, and optimize marketing and product strategies?”**

# Project Workflow
1. **Analyze data in jupiter notebook**
      - Data Import
      - Data exploration
      - Data cleaning
      - Connection to SQL Database
  
2. **Load the data from Python notebook into PostgreSQL**
      - Create a database in SQL
      - Run Python code to load data into SQL database
      - Answer Business Questions using SQL Queries 
      
3. **Connect the SQL Database to Power BI**   
      - Create an interactive dashboard in Power BI
